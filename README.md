# 穩卓國際 - 專業塑膠原料銷售官方網站

## 項目簡介

穩卓國際（Winzhuo International Trading Co., Ltd.）是香港穩卓國際貿易有限公司的專業塑膠原料銷售官方網站，采用純靜態 HTML + CSS 技術棧開發，支持中文/英文/越南文三語切換。

## 公司簡介

香港穩卓國際貿易有限公司成立於2009年，是一家專業從事塑膠原料貿易及國外品牌代理的企業。公司總部位於香港，在中國大陸、東南亞等地設有辦事處和倉儲中心。

多年來，我們始終堅持"質量第一，客戶至上"的經營理念，為客戶提供高品質的產品和專業的服務。我們與多家國際知名塑膠原料生產商建立了長期穩定的合作關係，代理銷售多個國際知名品牌的塑膠原料，同時與優質生產廠家合作開發定製化產品。

我們的產品廣泛應用於電子電器、汽車、家電、醫療、包裝等行業，服務全球500+客戶，以專業的服務和優質的產品贏得了客戶的信賴。

### 發展歷程

- **2008年**: 穩卓投資有限公司成立
- **2009年**: 穩卓國際貿易有限公司在香港成立
- **2010年**: 業務擴展，在中國大陸設立首個辦事處和倉儲中心
- **2012年**: 設立東莞辦公室，與代理品牌深度合作
- **2015年**: 設立廣州辦公室，市場拓展至東南亞
- **2018年**: 科卓實業（深圳）有限公司成立
- **2020年**: 設立惠州辦公室
- **2021年**: 創新發展，引入生物降解塑料產品線
- **2022年**: 增加彈性體事業部，創建品牌WinFlex
- **至今**: 持續成長，不斷創新

### 核心業務

專業代理全球優質塑膠原料，服務汽車、電子、醫療等多個行業：

1. **工程塑料**: 耐高溫/高強度/耐腐蝕
2. **通用塑料**: PP/PE/ABS/PVC等系列
3. **可降解塑料**: PLA/PHA
4. **熱塑性彈性體**: TPE/TPR/TPU/TPV/TPEE等系列（WinFlex自主品牌）
5. **特種材料**: PEEK/PPS 導電/阻燃/抗靜電/抗黃變/減震等系列

## 設計特色

### 色彩方案
- **主色調**: #2563eb (品牌藍)
- **深藍色**: #1a365d
- **背景色**: #f8fafc (淺灰)
- **文字色**: #111827 (深黑)
- **次要文字**: #6b7280 (中灰)
- **漸變色**: linear-gradient(135deg, #2563eb 0%, #1a365d 100%)

### 響應式斷點
- **移動端**: < 640px
- **平板**: 640px - 900px
- **桌面**: > 900px

### 動畫效果
- 平滑滾動 (scroll-behavior: smooth)
- 卡片懸停上移 (transform: translateY(-8px))
- 導航欄滾動漸變（背景變為 rgba(255,255,255,0.95)）
- 按鈕懸停效果
- 產品卡片動畫 (fadeInUp)
- 微信二維碼懸停顯示效果
